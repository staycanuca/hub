# Radio RO Kodi Plugin

This plugin allows you to listen to radio stations from different countries in Kodi.

## Features

*   Browse all radio stations with pagination
*   Listen to the most popular radio stations with pagination
*   Search for a specific radio station
*   Add stations to your favorites for quick access
*   Change the country for the radio stations in the settings
*   Select the API server for Radio Browser in the settings
*   The station list is always up-to-date, fetched directly from the Radio Browser API.
*   Detailed information for each station (website, tags, language, country, bitrate, codec) is available in the Kodi info dialog.

## How to install

1.  Create a zip file containing the following files and folders:
    *   `addon.xml`
    *   `main.py`
    *   `resources/settings.xml`
2.  Open Kodi and go to `Add-ons` -> `Install from zip file`.
3.  Select the zip file you created in step 1.
4.  The plugin will be installed and you can find it in `Add-ons` -> `Audio add-ons`.

## How to use

1.  Open the `Radio RO` plugin.
2.  You will see a menu with five options: "Most Popular", "All Stations", "Favorites", "Search" and "Settings".
3.  Select an option to browse the radio stations.
4.  When browsing "All Stations" or "Most Popular", you can use the "Next" and "Previous" buttons to navigate through the pages.
5.  To add a station to your favorites, open the context menu on a station and select "Add to Favorites".
6.  To remove a station from your favorites, open the context menu on a station in the "Favorites" list and select "Remove from Favorites".
7.  To change the country, go to "Settings" -> "General" and enter the desired country code (e.g., US, GB, DE).
8.  To select the API server, go to "Settings" -> "API Server Settings" and check the desired server. Only one server should be checked at a time.
9.  Select a station to start playing.
10. To view detailed information about a station, open the info dialog (usually by pressing 'i' on your keyboard or the info button on your remote).
